## Question 2: Factorial of a Number

### Problem Statement
Write a program that computes the factorial of a given number.

### Input
A single integer number.
```
8
```

### Expected Output
The factorial result printed as a single value.
```
40320
```

### Example
| Input | Output |
|-------|--------|
| 8 | 40320 |
| 5 | 120 |
| 0 | 1 |

### Concepts Covered
- Loops / Recursion
- Math module (`math.factorial`)

### Explanation
Factorial of a number n is the product of all positive integers from 1 to n.
For example: 8! = 8 × 7 × 6 × 5 × 4 × 3 × 2 × 1 = 40320
